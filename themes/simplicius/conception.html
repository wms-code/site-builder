STANDARD Conception Features

The template was designed by standard conception of classic websites. The template includes one homepage and one standardised subpage that can be configured by your web conception. It enables a high level of customisation and many universal ways to use it in a lot of variations at your business.

Homepage Description

The Homepage is designed by the consideration to use for standard website requirements that could carry out many of your needs. The main feature of template is high level of modifications and variations that you can set by the simple changing of several classes. It includes:

  • Header (logo & horizontal top menu)
  • Customisable background (combinations of 3 backgrounds)
  • Customisable page style (dark/light style)
  • Customisable Slider with or without light/dark description (great JS efect slideshow, black/white variation)
  • Unlimited columns combinations (1,2,3,4 column variations with variable line settings)
  • Unlimited box combinations (1,2,3,4 box variations with variable dimensions and black/white style)
  • Recent links in the full width box in black or white variation
  • Footer (classical footer same on every page)

Subpage Description

The Subpage is designed by the consideration to customisation of a different contents and components you may need. You can use the all elements same as the homepage like great slider or free combinations of boxes at the top or bottom of the page. It includes:

  • Header (logo & horizontal top menu)
  • Customisable background (combinations of 3 backgrounds)
  • Customisable page style (dark/light style)
  • Customisable Slider with or without light/dark description (great JS efect slideshow, black/white variation)
  • Customisable great picture with variable height and with or without description
  • Unlimited columns combinations at the top or bottom of the page (1,2,3,4 column variations with variable line settings)
  • Unlimited box combinations at the top or bottom of the page (1,2,3,4 box variations with variable dimensions and black/white style)
  • Unlimited sidebar box combinations that can be dependent by the light/dark page style or manully set to black/white style
  • Recent links in the full width box in black or white variation
  • Footer (classical footer same on every page)
